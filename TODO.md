# Open Talons — Roadmap

## ✅ v2.0 (Current)
- Multi-LLM core (12 providers)
- 20 crypto plugins ported from UFO Terminal
- Plugin SDK with definePlugin/defineTool/defineAgent
- Permission system (allow_always/ask/deny)
- Skills registry (OpenCode-style)
- Slash command framework
- Memory system per workspace
- Unified CLI with 22 agents

## 🔄 v2.1 (Next)
- [ ] Sub-agent orchestrator (master → child delegation)
- [ ] Streaming responses across providers
- [ ] Conversation history persistence
- [ ] MCP server integration
- [ ] Better error recovery in task loop
